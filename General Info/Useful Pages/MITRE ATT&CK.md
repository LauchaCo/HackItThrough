## What is?
__MITRE ATT&CK__ is a globally-accessible knowledge base of adversary tactics and techniques based on real-world observations. It is separated in different categories which are:
- Reconnaissance: Techniques used to obtain information about a target either actively or passively. This information will prove useful in the next stages of the pentest.
- Resource Development: Techniques such as, creating, purchasing, or compromising/stealing resources that can be used to support targeting. An example of this is using purchased domains to support [Command and Control](</General Info/Important concepts/Command and Control.md>) or stealing code signing certificates to help with Defense Evasion.
- Initial Access: Techniques to gain initial foothold within a network. Some examples are spearphishing
