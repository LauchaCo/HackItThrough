[[192.168.100.128 (Nagini Vmnet 3)]]
1. Enumerate ports with proxychains
```
sudo proxychains -sT -Pn -n -v -open --top-ports 500 10.10.0.128 -oG targeted
```
2. Seeing that the 445 port is open (SMB) we can use crackmapexec with proxychains to obtain information.
```
proxychains crackmapexec smb 192.168.100.130 2>/dev/null
```
This returns Windows 7 (It's interesting to know that SMB + Windows 7 is a bad combination because of eternal blue)
